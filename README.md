#  Calculator Project in C

##  Description
This is a basic calculator program developed using the C programming language.  
It performs fundamental arithmetic operations and is designed to strengthen core programming concepts.

---

##  Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- User-friendly input system

---

##  Technologies Used
- C Programming Language
- Standard Input/Output Functions

---

##  Concepts Applied
- Switch Case
- Loops
- Functions (if used)
- Conditional Statements
- User Input Handling

---

##  How to Run
1. Compile the program using a C compiler (like GCC)
2. Run the executable file
3. Enter numbers and choose the operation

---

##  Purpose
This project is created as a beginner-level program to practice logic building and understand how arithmetic operations work in C.
